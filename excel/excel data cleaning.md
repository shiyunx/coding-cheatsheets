# Excel Data Cleaning Methods

1. Check for duplicate

Select the column
Home > Conditional formatting > Highlight cells rules > Duplicate values

2. Extract name into first & last

Select the column
Data > Text to columns > Open convert the text to columns wizard
Check delimited > Next > Check space > Next > General > Finish

Text to columnsName will separate into First & Last

3. Delete blank rows

Select entire table and filter all columns
Check each column, deselect other options and select only blank cells
Note: Delete the blank rows only if all the columns are blank cells

Rows with some blank cells
Select entire table
Home > Find & Select > Go to special > Select blanks > Ok > Fill colour

Red highlighted blank cellsHome > Delete sheet rows

4. Remove spaces
Insert a new column > Trim formula > Enter

Trim formulaColumn J2 > Double click at the corner of a cell to auto correct the remaining cells
