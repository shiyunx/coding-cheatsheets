#### Cell
    $A$2: fixed column and row
    A$2: fixed row
    $A2: fixed column

##### Custom number format
    0: display zeros
    (use #.00 > 9.20)

#### Omit extra zeros displayed 
    (use #.## > 3.4)

#### Keyboard short cuts
    ctrl A > select entire table 
    ctrl Z > undo
    ctrl; > fixed date 
    ctrl shift; > fixed time
    = today() > current date
    = now() > current date and time
